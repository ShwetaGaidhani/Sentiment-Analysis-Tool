# Sentiment-Analysis-Tool
This project is a simple sentiment analysis system developed using Python and machine learning. The program analyzes user text and classifies it as positive, negative, or neutral using natural language processing techniques.

The system preprocesses text by cleaning it, removing stopwords, and converting it into numerical features using TF-IDF or Count Vectorizer. Machine learning models such as Logistic Regression and Naive Bayes are used to train the sentiment classifier.

This project was created as part of an Artificial Intelligence internship task.

Features
Text preprocessing and cleaning
Sentiment classification (positive, negative, neutral)
TF-IDF and Count Vectorizer support
Logistic Regression and Naive Bayes models
Interactive command line interface
Confidence score and probability output

Example Sentences
This product is amazing
The service was terrible
It is okay, nothing special
I really loved the quality
The experience was disappointing
